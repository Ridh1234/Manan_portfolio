# Manan Jain — Portfolio

Professional portfolio for **Manan Jain**, Chartered Accountant and private credit analyst based in Mumbai.

Built with React 19, TypeScript, and Vite. Cardless open sections sit on a full-page skyline photograph with glass panels for dense data blocks.

## Sections

- Hero — credential snapshot and positioning
- About — bio and at-a-glance tags
- Track Record — deal metrics, sectors, and structures
- Experience — career history
- Credentials — academic credentials and recognitions
- Skills — competencies and tools
- Contact — email, phone, and LinkedIn

## Development

```bash
npm install
npm run dev
```

```bash
npm run build
npm run preview
```

## Background image

Swap the skyline photo in `src/styles/variables.css` via `--site-bg-image` (files live in `public/backgrounds/`).
